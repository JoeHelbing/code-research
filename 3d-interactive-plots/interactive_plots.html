1. Sombrero Function Surface

The Sombrero (Mexican Hat) function is a classic example in signal processing and wavelet analysis. It's defined as the sinc function in radial coordinates, creating a characteristic shape with a central peak and rippling waves.

3. Parametric Surfaces - Torus & Helix

Parametric surfaces are defined by equations that express coordinates as functions of parameters. The torus (doughnut shape) is created by rotating a circle around an axis in 3D space. The cyan helix wraps around the torus, demonstrating how multiple 3D objects can be composed in a single interactive visualization.

Torus equations:
x = (R + r·cos(v))·cos(u)
y = (R + r·cos(v))·sin(u)
z = r·sin(v)
Helix equations:
x = R·cos(t)
y = R·sin(t)
z = t·0.5
